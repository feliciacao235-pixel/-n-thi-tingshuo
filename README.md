# 📚 Web Học Ngữ Pháp Nghe Nói 6

Web học tập tiếng Trung cho môn Ngữ pháp Nghe Nói 6, tập trung vào thành ngữ và cấu trúc ngữ pháp.

## 🎯 Tính năng

### 1. **Flashcard Thành Ngữ**
- Học thành ngữ với phát âm pinyin
- Xem nghĩa tiếng Việt và ví dụ minh họa
- Lật thẻ để xem chi tiết
- Đánh dấu "Đã nhớ" để theo dõi tiến độ

### 2. **Bài Tập Chọn Thành Ngữ**
- Chọn thành ngữ phù hợp cho từng câu
- 4 đáp án lựa chọn
- Giải thích chi tiết sau mỗi câu trả lời
- Tự động highlight đáp án đúng/sai

### 3. **Bài Tập Hoàn Thành Hội Thoại**
- Điền từ vào chỗ trống theo cấu trúc cho sẵn
- Dạng fill-in-the-blank dễ làm
- Gợi ý cấu trúc ngữ pháp cần sử dụng
- Kiểm tra tự động

### 4. **Theo Dõi Tiến Độ**
- Lưu kết quả học tập tự động
- Xem phần trăm hoàn thành mỗi unit
- Thống kê số câu đúng/sai
- Dữ liệu lưu trên trình duyệt

## 🚀 Cách Sử Dụng

### Cách 1: Mở trực tiếp
1. Mở file `index.html` bằng trình duyệt (Chrome, Firefox, Edge...)
2. Chọn Unit muốn học
3. Bắt đầu với Flashcard hoặc Luyện tập

### Cách 2: Chạy bằng Live Server (Khuyên dùng)
1. Cài đặt extension "Live Server" cho VS Code
2. Click chuột phải vào file `index.html`
3. Chọn "Open with Live Server"
4. Web sẽ tự động mở trên trình duyệt

## 📱 Hỗ Trợ Thiết Bị

- ✅ Desktop/Laptop
- ✅ Tablet
- ✅ Mobile (Responsive design)

## 🎨 Công Nghệ Sử Dụng

- **HTML5**: Cấu trúc trang web
- **CSS3**: Styling và animation
- **JavaScript**: Logic và tương tác
- **LocalStorage**: Lưu trữ tiến độ học tập

## 💾 Cấu Trúc File

```
.
├── index.html          # Trang chính
├── styles.css          # CSS styling
├── app.js              # JavaScript logic
├── data.js             # Dữ liệu thành ngữ và bài tập
└── README.md           # Hướng dẫn
```

## 🎯 Lộ Trình Học Đề Xuất

1. **Học Flashcard** → Xem và ghi nhớ thành ngữ
2. **Luyện Tập** → Làm bài không giới hạn để quen
3. **Kiểm Tra** → Test kiến thức và xem điểm số

## 📝 Ghi Chú

- Dữ liệu tiến độ được lưu tự động trên trình duyệt
- Xóa cache/cookies sẽ mất dữ liệu tiến độ
- Nên học tuần tự từ Unit 4 đến Unit 11
- Mỗi unit nên học Flashcard trước khi làm bài tập

## 🔧 Tùy Chỉnh

Để thêm thành ngữ hoặc bài tập mới, chỉnh sửa file `data.js`:

```javascript
// Thêm thành ngữ mới
IDIOMS_DATA[4].push({
    chinese: "新thành ngữ",
    pinyin: "xīn chéngyǔ",
    meaning: "Nghĩa tiếng Việt",
    example: "Ví dụ câu..."
});

// Thêm bài tập mới
EXERCISES_DATA[4].push({
    type: "idiom",
    question: "Câu hỏi...",
    options: ["A", "B", "C", "D"],
    correct: 0,
    explanation: "Giải thích..."
});
```
